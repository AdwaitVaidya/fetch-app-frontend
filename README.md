## Prerequisites

### Install Node JS
Refer to https://nodejs.org/en/ to install nodejs

## Quickstart


First, clone the repository:

```bash
git clone https://github.com/AdwaitVaidya/fetch-app-frontend.git fetch-app-frontend
cd fetch-app-frontend

# install the dependencies
npm install

# start the server
npm start
```
